# What is Rummage?
Rummage is an app that allows the user to add in select ingredients to find a cocktail recipe that includes what the user input. Another feature is the pantry, where you can store many ingredients and search those as well.  The idea is to find great cocktail recipes with the ingredients found in your home. 

### The Reason Behind it
Rummage is helpful for multiple reasons, It allows the user to utilize the ingredients they have readily available to craft a yummy cocktail. The app also helps the user find cocktail recipes that they would normally not try, and the main issue Rummage solves is the laziness aspect (or what we call had too many) aka stay in the comfort of your own home.
